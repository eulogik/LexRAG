# ⚖️ LexRAG — Legal Intelligence Terminal

**Enterprise-Grade Hybrid RAG for UAE & Indian Law, Taxation & Compliance**

This Space provides a minimal, beautiful chat interface for LexRAG.

- **Live Mode**: Requires a running LexRAG API server
- **Demo Mode**: Pre-loaded sample responses for key legal topics

## Links

## Usage

```bash
pip install lexrag
python -m uvicorn api.main:app --port 8000
```
